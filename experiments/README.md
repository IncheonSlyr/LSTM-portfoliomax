# Experiments

This folder contains optional exploratory notebooks or scripts that are not required by the main Streamlit portfolio dashboard.

## Institution GCN

`institution_gcn.py` is a small graph neural network experiment over the sample `institution_data.csv` dataset. It requires optional dependencies that are intentionally not part of the main dashboard requirements:

- `torch`
- `torch-geometric`
- `networkx`

Run it only after installing those packages in a separate environment.
